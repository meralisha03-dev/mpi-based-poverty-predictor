# Multidimensional Poverty Risk Predictor (India)

An interactive machine learning application that predicts multidimensional poverty risk across Indian states using socio-economic indicators.

## 🚀 Features
- Predict poverty risk (Low / Medium / High)
- Identify key contributing factors
- Provide policy-based solutions
- Compare states
- Interactive dashboard

## 🧠 Model Used
- Random Forest Classifier

## 📊 Dataset
- Source: NITI Aayog MPI 2023

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
